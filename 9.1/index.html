简述

CPython为开发者提供了轻松调用c代码的特性，之所以会有中特性，当然是为了高性能。

通常情况有三种方法可以在自己的Python代码中调用c写的函数，

  • ctypes
  • SWIG
  • Python/C API

最单的方法当属ctypes了，而且也是使用最多的方法。ctypes提供了和c语言兼容的数据类型和函数来加载动态库，因此在调用c函数时候，不需要对源文件做任何的修改。
